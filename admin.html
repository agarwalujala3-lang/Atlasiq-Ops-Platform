Security checklist

Core controls that keep the knowledge pipeline production-minded.

  • Secret key stays in `.env` only.
  • Rate limiting applied to generation route.
  • Source content length is capped before model calls.
  • Fallback responses avoid leaking raw backend errors.
  • Client exports do not expose the API key.
